# CMPM121HW5
Assignment 5

Start on the "MainMenu" scene. 

Controls:

> W - Move Forward

> S - Move Backwards

> A - Turn Left

> D - Turn Right

> Main Menu:
Use mouse to click on "Play" button to start the game.

Objective:
Collect the yellow cubes in the air to open doors. Collect all the yellow cubes to win the game.
